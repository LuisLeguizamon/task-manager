# task-management

### Description
* Create/edit/order tasks.

### Instalation
* Clone this repository
* composer install
* create a new .env file (use .env.example), configure a database
* php artisan migrate

### Development 
* cd task-management/
* php artisan serve

### Built With
* [Laravel](https://laravel.com/) - v10
* [MySQL](https://www.mysql.com/) - v8
* [PHP](https://www.php.net/) - v8.1
